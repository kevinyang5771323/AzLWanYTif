# 项目概述 java111

本项目是基于 SpringBoot 和 Vue 的前后端分离通用管理系统，包含智能代码生成器。系统支持单体版和多模块版本，集成了环境搭建、项目运行及代码生成器的使用教程。

# 技术栈

- 后端：SpringBoot
- 前端：Vue + ElementUi
- 数据库：MySQL 8
- 代码生成器：MybatisPlus

# 功能模块

## 系统角色与权限

- 用户管理：管理后台的登录用户和对用户授权
- 角色管理：分配菜单权限
- 菜单管理：动态管理页面上的路由
- 权限控制：包括后端和前端的功能访问控制，前端可控制到按钮级别

## 组织架构管理

- 部门管理
- 岗位管理

## 系统监控

- 日志管理：记录后台登录及所有操作
- 字典管理：统一管理常用数据分类

## 代码生成器功能

- 增删改查功能一键生成，支持一对多、多对多、多表同时操作
- 后端和前端的代码可根据需求一键生成
- 支持在代码生成界面上选择生成不同的页面组件，包括树形选择组件和弹窗选择组件

# 运行环境

- 后端：Java 1.8 或以上版本
- 前端：Node.js 10.0 或以上版本
- 数据库：MySQL 8.0 或以上版本
